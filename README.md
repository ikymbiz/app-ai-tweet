# 🚀 AI Chat & Tweet

ローカルブラウザ完結型で動作する、ペルソナ特化型の高機能AIチャット＆X(Twitter)連携アプリケーションです。
単一のHTMLファイルでありながら、複数LLMの切り替え、詳細なペルソナ（人物像）設定、AIによるツイート自動生成、ローカルDBへの履歴保存など、強力な機能を備えています。

![スクリーンショットのプレースホルダー](https://via.placeholder.com/800x400.png?text=App+Screenshot+Here)

## ✨ 主な特徴 (Features)

### 🤖 マルチLLM対応
*   **Google Gemini**, **OpenAI (ChatGPT)**, **Anthropic (Claude)** の主要APIに対応。
*   カスタムモデルの追加もUIから簡単に設定可能。
*   ストリーミング応答（Web Workers活用）による高速かつスムーズなチャット体験。

### 🎭 高度なペルソナ（プロフィール）管理
*   「基本情報」「思考・興味関心」「行動様式・生活スタイル」など、詳細なペルソナを設定可能。
*   設定したペルソナをシステムプロンプトとして組み込み、AIの口調や性格を完全にコントロール。
*   **AI自動入力機能**: 一部の情報を入力するだけで、残りのプロフィール項目をAIが推測して自動生成・補完します。

### 🐦 X (Twitter) 連携＆AIツイート生成
*   ブラウザから直接X（Twitter）へツイートを投稿可能（Bearer Tokenを使用）。
*   **AIツイート案生成**: ニュースのURLや今の気分を入力するだけで、設定したペルソナ（人物像）になりきった自然なツイート案をAIが140文字程度で生成します。
*   AIの回答を完了と同時に自動でツイートする「自動投稿モード」も搭載。

### 🔒 完全なプライバシーとローカル保存
*   サーバーレス設計。APIキーやチャット履歴、設定項目はすべてブラウザのローカルストレージ（IndexedDB）に安全に保存されます。
*   チャット履歴の検索、インポート / エクスポート機能。
*   チャット内容をMarkdownやJSON形式でローカルファイルとして保存可能（File System Access API利用）。

### 📱 優れたUI/UX
*   Tailwind CSSを利用したモダンでレスポンシブなデザイン。スマートフォンでも快適に動作。
*   PWA（Progressive Web App）対応。
*   Markdownレンダリングとコードハイライト表示対応。
*   AI生成中の画面スリープを防止するWake Lock API対応。

## 🚀 使い方 (Getting Started)

このアプリケーションはサーバーの構築やビルドプロセスを必要としません。

1.  このリポジトリをクローンするか、`index.html`（HTMLファイル）をダウンロードします。
2.  ダウンロードしたHTMLファイルをGoogle Chrome, Edge, Safariなどのモダンブラウザで開きます。

### 初期設定

1.  画面右上の **「⚙️（設定アイコン）」** をクリックして設定パネルを開きます。
2.  **「APIキー設定」** セクションに移動し、使用したいAIプロバイダー（Google, OpenAI, Anthropic）のAPIキーを入力します。
    *   *※入力したAPIキーはブラウザ内にのみ保存され、外部サーバーに送信されることはありません。*
3.  設定パネルを閉じると、すぐにチャットを開始できます。

## 💡 活用ガイド (Usage Guide)

### 1. ペルソナを作ってみよう
設定パネルの「プロフィール管理 (ペルソナ)」から「追加」ボタンを押します。
「氏名」や「職業」など少しだけ項目を埋め、上部の **「AI自動入力」** ボタンを押してみてください。AIが整合性の取れたリアルな人物像を自動で完成させます。
保存後、設定パネルの「現在のチャット設定」でそのプロフィールを選択すると、AIがその人物として振る舞い始めます。

### 2. AIにツイートを考えさせよう
チャット入力欄の右側にある **「Twitterアイコン」** をクリックします。
「AIにツイート案を作成させる」の入力欄に「今日のランチで行ったイタリアンが美味しかった」など適当な出来事を入力し、生成ボタンを押します。
現在設定されているペルソナの性格や口調を反映した、バズりやすいツイート案が瞬時に作成されます。

### 3. データのバックアップ
左上の **「ハンバーガーメニュー」** からサイドバーを開き、「Export」ボタンを押すことで、これまでのチャット履歴をすべてJSONファイルとしてバックアップできます。別のブラウザや端末に移行する際は「Import」から読み込んでください。

## 🛡 セキュリティと制限事項 (Security & Limitations)

*   **APIキーの管理**: 本アプリはクライアントサイドのみで動作します。APIキーの漏洩を防ぐため、共有のパソコン等で利用する場合は、利用後に設定画面からAPIキーを削除するか、シークレットウィンドウをご利用ください。
*   **Twitter APIの制限**: X(Twitter)への直接投稿機能は、CORS（Cross-Origin Resource Sharing）の制限を受ける場合があります。本番環境で確実に動作させるには、適切なCORSプロキシを設定するか、ブラウザの拡張機能を利用するなどの対策が必要になる場合があります。
